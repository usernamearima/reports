# 1. Overview
The **Linux Fundamentals Part 1** room is designed to introduce core concepts of the Linux operating system. It covers basic commands, file system navigation, file manipulation, permissions, and introductory shell usage. The room does not involve attacking a target but teaches foundational skills for system interaction.

---

# 2. Basic Command Line Navigation
Learners are guided through essential Linux commands to explore and interact with the filesystem:

- `pwd` – prints the current working directory  
- `ls` – lists directory contents  
- `cd` – changes the current directory  
- `man` – displays manual pages for commands  
- `clear` – clears the terminal screen

These commands form the basis for navigating any Linux environment.

---

# 3. File and Directory Operations
The room demonstrates standard file operations using core utilities:

- Creating files: `touch filename`  
- Reading files: `cat filename`, `less filename`  
- Copying files: `cp source destination`  
- Moving/renaming files: `mv source destination`  
- Removing files: `rm filename`  
- Creating directories: `mkdir directoryname`  
- Removing directories: `rmdir directoryname`

These operations establish basic file management proficiency.

---

# 4. Text Editors Introduction
The room introduces basic text editing using common editors:

- `nano` – simple command‑line editor  
- `vim` – powerful modal editor (introduced at a basic level)

Learners practice editing and saving text files using these editors.

---

# 5. Permissions and Ownership
Linux file permissions are explained, including read (r), write (w), and execute (x) permissions. The room covers:

- Viewing permissions: `ls -l`  
- Changing permissions: `chmod`  
- Understanding owner and group concepts

This section prepares learners for secure management of files and scripts.

---

# 6. File Searching and Processes
Fundamental tools for searching and process overview include:

- Searching files: `grep pattern filename`  
- Find files by name: `find /path -name filename`  
- Viewing processes: `ps aux`  
- Filtering process output: `top`, `htop`

These utilities assist with basic system inspection and debugging.

---

# 7. Networking Basics
Introductory networking commands are covered:

- `ifconfig` / `ip a` – displays interface configuration  
- `ping hostname` – tests connectivity  
- `netstat` / `ss` – shows active sockets

These commands introduce networking visibility within Linux.

---

# 8. Impact
Mastering Linux fundamentals enables:

- Efficient interaction with Linux systems  
- Confident use of the command line environment  
- Foundation for advanced topics such as scripting, security, and administration

These skills are essential for any cybersecurity professional or system administrator.

---

# 9. Key Takeaways
- Linux command line skills are foundational for cybersecurity and DevOps roles  
- Understanding file operations and permissions is critical for secure system management  
- Basic navigation and tool usage improve productivity in any Linux environment  
- Familiarity with process and network tools facilitates troubleshooting and monitoring
