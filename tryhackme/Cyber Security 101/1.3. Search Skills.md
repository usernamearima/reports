# 1. Overview
The Search Skills room is part of the **Cybersecurity 101** learning path and focuses on developing efficient search techniques for cyber research. It does not involve attacking a machine but teaches how to find relevant information using search engines, specialized tools, social media, and documentation. :contentReference[oaicite:0]{index=0}

---

# 2. Core Learning Objectives
The room’s primary goals include:

- Evaluating information sources for credibility  
- Using search engines effectively with advanced operators  
- Exploring specialized search engines for cyber data  
- Reading and referencing technical documentation  
- Leveraging social media and news outlets for intelligence gathering :contentReference[oaicite:1]{index=1}

---

# 3. Search Technique Fundamentals
Learners are introduced to best practices for searching:

- Understand source reliability (authoritativeness, evidence, bias)  
- Apply advanced operators (e.g., exact phrases, site restrictions, exclusions, filetype filters)  
- Use specialized search engines such as Shodan (Internet‑connected devices), Censys (hosts and certificates), VirusTotal (file/URL scanning), and Have I Been Pwned (email breach checks) :contentReference[oaicite:2]{index=2}

---

# 4. Specialized Search Engines Coverage
The room covers practical use of security‑specific search tools:

- **Shodan:** Searches for exposed devices and services  
- **Censys:** Focuses on hosts, sites, and certificate data  
- **VirusTotal:** Combines multiple antivirus engines to assess files and URLs  
- **Have I Been Pwned:** Identifies breached credentials via email address queries :contentReference[oaicite:3]{index=3}

---

# 5. Technical Documentation Search
Participants learn how to locate and use official documentation:

- Linux manual (man) pages for command and system reference  
- Microsoft Technical Documentation for Windows commands and utilities  
- Official product docs (e.g., Snort, Apache, PHP) for detailed behaviour and configuration guidance :contentReference[oaicite:4]{index=4}

---

# 6. Social Media & OSINT
The room emphasises using social platforms and outlets for research:

- Search professional networking sites for personnel information  
- Use social platforms to find context relevant to security evaluations  
- Recognise how shared personal info may relate to security questions or investigations :contentReference[oaicite:5]{index=5}

---

# 7. Impact
Mastering search skills equips learners to:

- Quickly locate technical answers from authoritative sources  
- Find vulnerability information, flags, or exploit details during analysis tasks  
- Validate threat intelligence and cross‑reference multiple independent sources  
- Reduce time spent stuck on research by using optimized search strategies :contentReference[oaicite:6]{index=6}

---

# 8. Key Takeaways
- Effective search strategies are essential for security research  
- Advanced search operators significantly improve result relevance  
- Specialized search engines provide security‑specific insights unavailable via general web search  
- Reliable documentation lookup accelerates learning and troubleshooting  
- Social media and breach databases are indispensable in modern cyber investigations :contentReference[oaicite:7]{index=7}

# SUMMARY
The Search Skills room on TryHackMe teaches foundational information‑gathering techniques critical for cybersecurity work. It emphasises using advanced search features, exploring domain‑specific search engines, and understanding how to evaluate and validate information from multiple sources. This skill set enhances threat intelligence, incident response, and general research effectiveness.

Flow: Introduction → Search Best Practices → Advanced Operators → Specialized Search Engines → Documentation Search → Social Media Exploration → Knowledge Application
